
A multi-condition MetaTrader 5 sell-signal indicator combining two independent setups: an "Upper Band" reversal (price touching/tolerating the Bollinger upper band) and a "Middle Band" breakdown (price rejecting the midline with downward slope). Each setup chains up to 10 configurable filter conditions — RSI overbought/decay checks, candle-high limits, RSI percentage decay, and current/higher-timeframe ZigZag swing-tolerance confirmation — combinable via AND/OR toggles. On confirmation it draws sell arrows and optional shaded signal zones on the chart, plus optional higher-timeframe ZigZag level lines, with popup/sound alerts fired once per new closed bar. Scans historical bars on load and evaluates only the latest bar in real time for efficiency; the buy-side counterpart is stubbed out for a future release.
