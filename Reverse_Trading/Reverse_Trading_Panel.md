
A draggable, resizable on-chart trading panel for MetaTrader 5, built specifically for XAUUSD (Gold). Provides one-click BUY/SELL buttons with configurable lot size and stop-loss (in points), plus a "Reverse Trade" button that closes the current position and opens an opposite one at an auto-calculated volume (last trade volume × user-defined multiplier). The reverse volume field can be manually overridden, and the panel tracks trade state live via OnTick. Includes full mouse-driven drag/resize handling with chart-scroll locking, and a close button to remove the panel. Built using MQL5's native object system (OBJ_BUTTON, OBJ_EDIT, OBJ_RECTANGLE_LABEL) and the CTrade class for order execution.
