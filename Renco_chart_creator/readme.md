
 A MetaTrader 5 Expert Advisor that generates two independent ATR-based Renko charts simultaneously from a single source symbol, each with its own timeframe, ATR period, and brick-size multiplier. Uses MQL5's Custom Symbol API to create synthetic Renko instruments and opens live charts for them, with TradingView-style reversal logic (bricks reverse only after a double-threshold move). Brick history is rebuilt from historical bars on startup, then updated live tick-by-tick via OnTick. Supports optional wick rendering and configurable brick history limits. Built around a reusable CRenkoGenerator class, allowing easy extension to more simultaneous Renko streams.
