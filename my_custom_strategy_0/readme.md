
A MetaTrader 5 on-chart indicator that combines EMA crossover, Bollinger Bands mean-reversion, and a Pine Script–style EMA-smoothed momentum filter to generate buy/sell arrow signals on XAUUSD. A signal fires when price closes outside the Bollinger Bands (or the fast/slow EMA crosses) while momentum confirms direction, with the arrow offset scaled to candle range. Optional debug lines can plot the EMA and Bollinger buffers directly on the chart for visual verification. Includes a built-in alert system that triggers a popup once per new bar when a fresh buy or sell signal appears. Rebuilds its signal history from CopyRates/CopyBuffer on every calculation pass rather than using incremental buffer updates.
